Projects

Bringing The Missing Home: A Comprehensive Identification System ⧉

  • The objective of this project is to provide a platform that enables users to input a video file, containing footage of a missing person, along with a photograph of the same individual.

  • Upon receiving this input, the system will analyze the video file and return the timestamp of the frame that features the most clear and identifiable image of the missing person as per the maximum matching of the person. The threshold for similarity or cosine score was 0.9.

  • Stack used: Python, SQL, ReactJs, JavaScript, APIs, Flask, Machine Learning, Node.js
